# Explore モード観察ノート

`word-antipode explore` で得られた意味ベクトルの層遷移可視化から読み取れた現象と、その transformer 的解釈をまとめる。

## 実験設定

| 項目 | 値 |
|---|---|
| 日付 | 2026-05-16 |
| モデル | `cyberagent/open-calm-small` (117M, 12層, 隠れ次元768) |
| 単語数 | 11語 |
| 中心化 | あり（各層内で平均ベクトル除去・再L2正規化） |
| 投影 | PCA 2次元 |

### 投入単語

```
愛, 憎しみ, 光, 闇, 生, 死, 始まり, 終わり, 喜び, 悲しみ, 武蔵高校地学部
```

意図: 対義概念ペア5組（愛↔憎しみ、光↔闇、生↔死、始まり↔終わり、喜び↔悲しみ）に、明らかに性質の異なる固有名詞句（武蔵高校地学部）を1つ混ぜた構成。

## 観察された3つの現象

### 現象1: 八の字（ループ）軌跡

11単語すべてが、層を進むにつれ2D PCA上で**直線ではなく八の字状に湾曲する軌跡**を描いた。

#### 解釈

transformer の residual stream が層ごとに違う方向の修正を加えていく結果。各層の役割が異なるため、軌跡は折れ曲がる:

- 浅い層（0-3）: トークン表面情報 → 統語へ
- 中間層（4-8）: 統語 → 抽象意味へ
- 深い層（9-11）: 抽象意味 → 出力準備へ

これらの方向が直交しないため、2D投影では曲線になる。**transformer 固有の本質的性質**であり、モデルサイズによらず観察される（Belrose et al. 2023, Tuned Lens 等）。

### 現象2: 11層→12層で初期位置付近へ回帰

最終層（12）で全単語が層0付近の座標に戻ってきた。これが本実験で最も重要な観察。

#### 解釈: tied embeddings と logit lens collapse

decoder-only LM の構造的必然:

1. 出力時の語彙確率は「最終隠れ状態 × 埋め込み行列の転置」で計算
2. 多くのモデル（OpenCALM含む）は**入力埋め込みと出力ヘッドが共有**（tied embeddings）
3. つまり最終層の出力は**入力埋め込みと同じ空間に戻す必要がある**

```
層0    : 入力埋め込み空間（トークン表面）
層1-11 : 抽象意味空間で計算
層12   : 次トークン予測のため、再び埋め込み空間に「翻訳し直す」
        ← ここで初期位置付近に戻る
```

#### 実用的含意

**純粋な意味解析には中間層（層7-10）の隠れ状態が最も適している**。
最終層は「次トークン予測のための変換」に特化しており、意味の抽出には不向き。

→ 本プロジェクトが `layer_fraction=0.6`（small=第7層、12層モデルなら7層目）をデフォルトとしているのはこの観察と整合的。

### 現象3: 「武蔵高校地学部 + 憎しみ」 vs 「その他9語」の二群分離

PCA上で全11単語が2つの群に分離。一見奇妙な組み合わせ。

#### 解釈: トークン化由来のアーティファクト

仕様4.1により**多トークン語は最終トークンの隠れ状態**を採用。トークン化を推定すると:

| 単語 | 推定分解 | 最終トークン | 性質 |
|---|---|---|---|
| 光・闇・愛・生・死 | 単一 | 漢字そのもの | 意味確定 |
| 喜び・悲しみ | 喜/び, 悲し/み | び, み | 無意味な仮名 |
| 憎しみ | 憎し/み | み | 無意味な仮名 |
| 始まり・終わり | 始ま/り, 終わ/り | り | 無意味な仮名 |
| 武蔵高校地学部 | 武蔵/高校/地学/部 | 部 | 汎用語、文脈依存 |

#### 二群分離の主因（推定）

PCA第1主成分が拾った最大の散らばり方向は、**意味の極性（ポジ/ネガ）ではなく「最終トークンの意味希薄度」または「複合語性」**だった可能性が高い。

`武蔵高校地学部` は7文字の固有名詞句で最も多トークン、`憎しみ` も「み」が最終で意味が薄い → この2語が「最終トークンが文脈なしでは意味を持たない」群として共通の象限に来た。

#### 教訓

- **「対義語が両極に分かれる」のはモデルが意味を捉えている証拠** だが、それ以前にトークン化が安定した単語セットを使う必要がある
- 1漢字語のみで実験すれば、PC1が意味極性をより素直に反映する可能性
- 固有名詞句のような複合語は、軌跡解析よりも「異常検知」的に使える

## 全体所感

| 観察 | 由来 | モデルを大きくすると |
|---|---|---|
| 八の字軌跡 | transformer本質 | 形は変わるが本質は同じ |
| 最終層の回帰 | tied embeddings | ほぼ全モデル共通 |
| 二群分離 | トークン化 | トークナイザ改善で緩和される可能性 |

**open-calm-small (117M)** での今回の観察は「LLMが意味を捉えていない」というよりも「**意味を捉える層と出力に最適化される層の役割分担**」が綺麗に見えた、と読むのが正しい。

層7前後で意味解析するという仕様の妥当性が視覚的に裏付けられたことが、本実験の最大の収穫。

## 次の検証

1. **より大きいモデル**（`cyberagent/open-calm-medium`, 400M, 24層）で同じ11単語を可視化し、軌跡形状と群分離パターンを比較
2. **1漢字語のみ**（光/闇/生/死/愛/憎/喜/悲/始/終 など）で実行し、トークン化の影響を排除した状態でのPC1を観察
3. 中心化なし（`--no-center`）と比較し、層深度ドリフトが除去できているかを確認
